# TOOLS.md - Notes outils utilisateur (modifiable)

Ce fichier est pour _vos_ notes sur les outils externes et les conventions.
Il ne définit pas quels outils existent ; OpenClaw fournit les outils intégrés en interne.

## Exemples

### imsg

- Envoyer un iMessage/SMS : décrivez qui/quoi, confirmez avant l'envoi.
- Préférez les messages courts ; évitez d'envoyer des secrets.

### sag

- Synthèse vocale : spécifiez la voix, l'enceinte/pièce cible, et si vous voulez le streaming.

Ajoutez tout ce que vous voulez que l'assistant sache sur votre chaîne d'outils locale.
